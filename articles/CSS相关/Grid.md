# CSS - Grid Layout

CSS 的核心布局方案 —— 网格布局。

采取二维网格概念来布局内容，有一组可推测结果的尺寸控制方法将当前布局空间划到行和列上。

这是一项非必须的，但必须存在的方案。目前，Grid 设计的很繁琐（即便它很强），主要表现在极其零散的语法和语句。因此，Grid 不会像 Flex 一样地普及，况且 Flex 简单又足够强大（Flex 的多级嵌套能实现各类布局）。
