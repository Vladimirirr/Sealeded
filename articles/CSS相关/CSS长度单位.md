# CSS 长度单位

## 绝对长度

物理意义上的长度，描述的是客观真实的物体的外观信息。

1. `cm` 厘米
2. `mm` 毫米
3. `Q` 毫米的 25%
4. `in` 英寸
5. `pc` 十二点活字
6. `pt` 点
7. `px` 像素，物理显示器的像素点

## 相对长度

会随着参考物变化而变化的长度。比如父字体大小、视口大小、等等。

1. `em` 相对父元素的字体大小（对于 font-size），相对自己的字体大小（其他 CSS 规则）
2. `ex` 字符`'x'`的高度
3. `ch` 字符`'0'`的宽度
4. `rem` 根元素的字体大小
5. `lh` 元素的 line-height
6. `vw` 视口宽度的 1%
7. `vh` 视口高度的 1%
